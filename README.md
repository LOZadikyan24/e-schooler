## 📋 Description

<p align="center">
<strong>E-Schooler</strong> is an educational platform that combines learning and testing in a single application.  
It provides structured content, practical exercises, and automatically generated tests based on a categorized question bank.  

💡The goal is to create a more interactive and efficient way for students to learn and evaluate their progress.
</p>

## 🚀 Features

<p align="center">
• Interactive learning content <br>
• Practice exercises and homework tasks <br>
• Automatic test generation (20 questions) <br>
• Question bank with categories (minimum 30 questions) <br>
• Weighted scoring system <br>
• Automatic grading (Bulgarian 2–6 system) <br>
• Performance statistics and analysis
</p>
